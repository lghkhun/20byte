# Device Endpoint

## GET `/device/info`

Ambil info perangkat WhatsApp aktif.

## GET `/device/qr-status`

Cek status QR pairing.

## POST `/device/generate-qr`

Generate QR pairing baru.

## POST `/device/generate-qr-link`

Generate link pairing.

## POST `/device/logout`

Logout device WhatsApp dari business aktif.
